# 开发工作流

## 问题说明

Expo 开发服务器在 Web 平台上无法正确加载路由（显示 "Welcome to Expo"），但静态导出工作正常。

## 临时解决方案

使用静态导出 + 本地服务器进行开发：

### 1. 构建静态文件
```bash
npm run build:web
```

### 2. 启动本地服务器
```bash
npm run serve
```

### 3. 访问应用
打开浏览器访问：`http://localhost:3000`

## 开发流程

1. **修改代码**
2. **重新构建**：`npm run build:web`
3. **刷新浏览器**查看更改

## 注意事项

- 每次修改代码后需要重新构建
- 构建时间约 20-30 秒
- 这是临时方案，后续可以尝试修复开发服务器问题

## 已验证工作的功能

✅ 路由系统正常
✅ 所有页面都能正确导出
✅ TypeScript 编译正常
✅ Supabase 配置正确
