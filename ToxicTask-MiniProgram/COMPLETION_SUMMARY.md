## 🚀 如何在微信开发者工具中运行

### 步骤 1：构建项目
```bash
cd ToxicTask-MiniProgram
npm install
npm run build:weapp
```

### 步骤 2：打开微信开发者工具
1. 下载并安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具
3. 选择"导入项目"

### 步骤 3：配置项目
- **项目目录**: `D:\ToxicTask\ToxicTask-MiniProgram\dist`
- **AppID**: 选择"测试号"（或输入你的真实 AppID）
- **项目名称**: ToxicTask

### 步骤 4：开发设置
在"详情" -> "本地设置"中勾选：
- ✅ 不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书
- ✅ 启用调试

### 步骤 5：测试功能
- 查看底部 TabBar（3 个带图标的 Tab）
- 测试登录/注册功能
- 创建任务
- 查看耻辱墙和个人资产
