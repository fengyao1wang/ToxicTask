# 成就系统测试指南

## 快速测试方法

### 时间管理类

#### 1. 早起的鸟 ✅
**测试步骤**：
1. 将系统时间调整到早上7点
2. 创建一个任务（押注10币，1小时后截止）
3. 在截止前完成任务
4. 查看成就页面，应该解锁"早起的鸟"

#### 2. 早起的虫 ✅
**测试步骤**：
1. 将系统时间调整到早上7点
2. 创建一个任务（押注10币，1小时后截止）
3. 等待任务超时失败
4. 查看成就页面，应该解锁"早起的虫"

#### 3. 零点战神 ✅
**测试步骤**：
1. 将系统时间调整到凌晨0点-2点之间
2. 创建任何任务
3. 立即查看成就页面，应该解锁"零点战神"

#### 4. DDL战神 ✅
**测试步骤**：
1. 创建一个任务（押注10币，10分钟后截止）
2. 等待到截止前4分钟
3. 完成任务
4. 查看成就页面，应该解锁"DDL战神"

#### 5. 消失的爱人 ⚠️
**测试步骤**：
1. 创建一个任务并让它失败
2. 关闭小程序
3. 24小时后再打开小程序
4. 查看成就页面，应该解锁"消失的爱人"
**注意**：需要真实等待24小时，或修改本地存储的 `last_active_${userId}` 时间戳

#### 9. 梭哈大师 ✅

#### 10. 华尔街之羊 ✅

## 调试技巧

### 使用调试面板
首页有调试面板，可以：
- **推进一天**：模拟时间流逝，用于测试需要多天的成就
- **重置日期**：恢复到真实日期

### 修改本地存储
可以通过微信开发者工具的 Storage 面板直接修改：
- `toxictask_transactions`：交易记录
- `toxictask_tasks`：任务列表
- `last_active_${userId}`：最后活跃时间

### 查看日志
在控制台搜索：
- `[AchievementStore]`：成就系统日志
- `[Info] 成就已解锁`：成就解锁提示

## 常见问题

### Q: 成就没有解锁？
A: 检查以下几点：
1. 确认触发条件是否完全满足
2. 查看控制台是否有错误日志
3. 确认成就检测是否被触发（查看日志）
4. 某些成就需要历史数据（如交易记录）

### Q: 如何重置成就进行测试？
A: 在微信开发者工具的 Storage 面板中：
1. 清除 `toxictask_user_achievements`
2. 刷新小程序

### Q: 如何快速获得大量尊严币？
A: 在 Storage 面板中修改 `toxictask_profiles` 中的 `dignity_coins` 字段
