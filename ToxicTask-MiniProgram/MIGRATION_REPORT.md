## 🚀 如何运行

### 1. 安装依赖
```bash
cd ToxicTask-MiniProgram
npm install
```

### 2. 构建项目
```bash
npm run build:weapp
```

### 3. 在微信开发者工具中打开
- 导入项目
- 选择 `ToxicTask-MiniProgram/dist` 目录
- 使用测试号或真实 AppID
